# Placeholder Icon

This is a placeholder icon. For production, create proper icons:

- **Windows**: Use an .ico file (16x16, 32x32, 48x48, 256x256)
- **macOS**: Use an .icns file or PNG (16x16@1x, 32x32@2x for Retina)
- **Linux**: Use PNG files

You can convert the SVG using tools like:
- ImageMagick: `convert tray-icon.svg -resize 256x256 tray-icon.png`
- Online converters
- Design tools like Figma, Sketch, or Adobe XD

For now, the app will show a warning if the icon is missing, but will still function.
